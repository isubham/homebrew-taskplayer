Three ways to show "how much time went to each task" in the sessions list — right now the list under the bar is purely chronological, one row per session, and the only way to see two sessions of the same task add up is to hover both colored blocks on the bar above. Same sample day throughout: two migration sessions, a standup, an afternoon migration block, a docs session, and one still running.

1 · Per-task totals, replacing the chronological list

One row per task instead of one row per session: dot, name, session count, total time for the day. Cleanest read of "where did today go" — but exact start/end times per session are gone from this list; they still live in the bar's hover tooltip above.
